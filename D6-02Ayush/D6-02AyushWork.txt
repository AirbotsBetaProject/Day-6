1. Wrote a code to test the xml files generated by fischer face classifier on random test subjects.
2. Tested 10 different xml files for 3 emotions - anger, sadness and happy.
3. The mean predicting accuracy for all files was 87 %.
4. The model mainly predicted emotions "anger" and "sadness" upto good extent but did not predict happiness at all.
5. Output files are named as : - <xml_file_no.>_<image_no.>_<predicted_emotion>.jpg .
